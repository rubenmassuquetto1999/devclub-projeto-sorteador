# Sorteador de Números 🎲

Um aplicativo web simples e responsivo que permite sortear um número aleatório dentro de um intervalo definido pelo usuário.

## 🚀 Funcionalidades

* **Sorteio Personalizado:** O usuário define os valores mínimo e máximo para o sorteio.
* **Validação Inteligente:** O sistema alerta diretamente na tela caso o usuário tente inserir um valor mínimo maior ou igual ao valor máximo, evitando falhas matemáticas.
* **Interface Dinâmica:** Os resultados e mensagens de erro são exibidos em componentes visuais customizados, dispensando o uso de pop-ups nativos do navegador.
* **Design Moderno e Responsivo:** Interface estilizada com background em degradê e design adaptável para funcionar perfeitamente em telas de computadores, tablets e smartphones.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do projeto.
* **CSS3:** Estilização de componentes, Flexbox para alinhamento e Media Queries para responsividade.
* **JavaScript:** Lógica de manipulação do DOM, escuta de eventos (Event Listeners) e funções matemáticas (`Math.random`, `Math.floor`, `Math.ceil`).

## ⚙️ Como usar

1. Faça o download ou clone os arquivos do projeto.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estão agrupados no mesmo diretório.
3. Garanta que exista uma pasta chamada `assets` na raiz do projeto contendo as imagens referenciadas (`sort 1.png` e `DADO.png`).
4. Abra o arquivo `index.html` em qualquer navegador web atual.
5. Insira os números nos campos "entre" e "e", e clique em **Sortear**.
